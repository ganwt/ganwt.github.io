五、関数 range()

1.range()

  • 指定された範囲の整数を順に生成

  • rangeのバリエーション

    • range( end )
    • range( start，end )
    • range( start，end, step )

  • end は省略できない

  • start が省略された場合，start = 0

  • step が省略された場合， step = 1

  • 生成される値には end は含まれない

  • step の値が負の場合には，値を減少させる

2.関数 rangeの使用例

  1. range( end )

  2. range( start，end )

  3. range( start，end, step )

  4. range( start，end, step )

    • step の値が負の場合には，値を減少させる

  5. rangeそのものは関数であって値を持たない

  6. listの生成

    • 関数 rangeを使ったリストの生成

  7. breakとcontinue

    • break
      • ループを終了
    • continue
      • 繰返しを スキップ
